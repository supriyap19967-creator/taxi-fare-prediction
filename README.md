
# Taxi Fare Prediction (End-to-End ML Project)

## Problem
Predict NYC taxi fare using trip features.

## Dataset
Kaggle – NYC Taxi Fare Prediction

## Steps
- Data cleaning & feature engineering
- Train Keras regression model
- Evaluate with MAE(Mean absolute error)
- Save trained model

## Tech Stack
Python, Pandas, Scikit-learn, TensorFlow (Keras)

## Result
MAE ≈ 0.25
